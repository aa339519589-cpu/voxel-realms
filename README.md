# VOXEL//REALMS

一个运行在浏览器中的原创体素沙盒。世界由种子确定，可探索、采集、建造并永久保存到本机。

## 功能

- 基于 Three.js 的全屏体素渲染、区块流式加载与隐藏面网格生成
- 平原、森林、沙漠、高山、水域、洞穴、矿脉、树木、仙人掌与遗迹
- 固定步长玩家物理、碰撞、跳跃、冲刺、潜行、游泳与创造飞行
- 方块挖掘、放置、粒子、程序化材质音效、九格快捷栏与生存合成
- 生命、饥饿、氧气、坠落伤害、昼夜、天气、动物与夜间敌对生物
- 桌面 Pointer Lock 控制与触屏摇杆、视角、跳跃、挖掘、放置控制
- IndexedDB 世界存档、自动保存、JSON 导入导出与脏数据修复
- 生存/创造模式、世界管理、设置、调试信息和命令控制台

## 本地运行

需要 Node.js 22.12 或更新版本。

```bash
npm ci
npm run dev
```

生产构建和测试：

```bash
npm test
npm run build
```

## 操作

| 操作 | 键位 |
| --- | --- |
| 移动 | `W` `A` `S` `D` |
| 跳跃 / 上升 | `Space` |
| 冲刺 | `Ctrl` |
| 潜行 / 下降 | `Shift` |
| 挖掘 / 放置 | 鼠标左键 / 右键 |
| 快捷栏 | `1` - `9` 或滚轮 |
| 背包与合成 | `E` |
| 丢弃一件 | `Q` |
| 创造飞行开关 | 双击 `Space` |
| 调试信息 | `F3` |
| 命令控制台 | `/` |
| 暂停 | `Esc` |

可用命令：`/time day|night`、`/weather clear|rain`、`/gamemode survival|creative`、`/tp x y z`、`/seed`。

## 数据与隐私

存档默认仅保存在当前浏览器的 IndexedDB 中，不上传到服务器。清理网站数据会删除本地存档，重要世界请从“世界存档”中导出 JSON 备份。
